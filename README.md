# CivicDesk 🚦

**A mobile app that empowers communities to report and track civic infrastructure issues.**


## 🎯 What is CivicDesk?

CivicDesk is a citizen-centric mobile application that bridges the gap between communities and local authorities. It allows residents to report civic problems like potholes, broken streetlights, garbage accumulation, and overgrown weeds with just a few taps.

## 🤔 Why CivicDesk?

Civic issues often go unreported or unnoticed until they become serious problems. CivicDesk makes it easy for citizens to:
- Document and report issues instantly with photos and location
- Track the progress of their reports in real-time
- Support community reports to help prioritize urgent issues
- Hold authorities accountable through transparent status tracking

By giving everyone a voice, CivicDesk helps build cleaner, safer, and better-maintained communities.

## ✨ Key Features

- 📸 **Quick Reporting**: Capture issues with photos and automatic location tagging
- 👥 **Community Feed**: See what others in your area are reporting
- 📊 **Status Tracking**: Monitor your reports from submission to resolution
- 👍 **Upvoting**: Support important community issues
- 🗺️ **Map View**: Visualize problems across your neighborhood
- 🌐 **Bilingual**: Available in English and Hindi (हिंदी)
- 🔒 **Anonymous Option**: Report issues privately if needed

## 🛠️ Built With

- React Native + Expo
- TypeScript
- TailwindCSS (NativeWind)

## 🚀 Getting Started

### For Developers

1. **Clone and install**
   ```bash
   git clone https://github.com/S-Munawar/CivicDesk.git
   cd CivicDesk
   npm install
   ```

2. **Run the app**
   ```bash
   npm start
   ```

3. **Test on device**
   - Install Expo Go on your phone
   - Scan the QR code to open the app
